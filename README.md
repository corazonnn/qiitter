## 概要
日々の学習の成果をソースコードと一緒に記録し、グラフ化するサービスです。
TwitterやQiitaはアウトプットの場としては素晴らしいですが、それらツイートや記事は**点**としてのみ存在し、その人の**ストーリー性**が感じられません。そこでソースコードとともに学習を記録し、それらの点を繋ぎグラフ化することで視覚化させます。
